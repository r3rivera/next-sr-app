# Context & Goal:
The banner should promote home discovery across seasons with a warm, lifestyle-driven feel. It should balance aspirational visuals with clear messaging and a strong call-to-action.

## Creative Direction:

### Style: Modern, minimal, premium real estate aesthetic
**Mood:** Warm, inviting, seasonal versatility (autumn tones, soft gradients, natural light)

**Layout:** Split or asymmetrical layout with image on one side and text on the other
Visual Hierarchy: Clear headline → supporting text → CTA

### Content Requirements:

**Headline:** “Find Your Home in Every Season”
**Subtext:** Short supporting line about comfort, lifestyle, or investment
**CTA Button:** “Browse Listings” or “Get Started”
**Sub-content**: 2–3 small circular image thumbnails showing interior spaces (living room, kitchen, bedroom)

### Design Elements:

Large hero image of a modern house
Soft gradient or seasonal color blocks in the background
Rounded corners or card-style container for the hero section
Clean sans-serif typography
Subtle shadows for depth
Consistent spacing system (8px or 4px grid)

### UX & Responsiveness:

Mobile-first design
Stack layout on smaller screens (image above text)
Maintain readable typography and tappable CTA
Ensure accessibility (contrast, font size, button clarity)

### Technical Output Requirements:

Provide structure that maps directly to HTML (sections, divs, classes)
Use semantic naming conventions (e.g., .hero, .hero-content, .cta-button)
Suggest CSS approach (Flexbox or Grid)
Include notes for responsive breakpoints

### Deliverables:

Banner concept description
Suggested HTML structure following the CSS guidelines provided in the project
